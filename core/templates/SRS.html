Kindergarten Management System

This interactive dashboard provides a dynamic overview of the Software Requirement Specification (SRS). It translates the formal document into an explorable format, clarifying the system's scope, user roles, and core functionalities.

Project Scope

A web-based application to manage admissions, academics, and communication, featuring distinct portals for Admins, Teachers, and Students.

Key Objectives

Automate core administrative and academic workflows, streamline communication, and provide a centralized platform for all school operations.

User Roles

The system is built around three primary user roles: **Admin** for system oversight, **Teacher** for academic management, and **Student/Guardian** for access to information.
